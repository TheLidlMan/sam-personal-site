# Sam Matthews personal site

Production static personal website for Sam Matthews.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

- Built with Vite + React.
- Public-safe content only.
- No private address, student number, phone, or private life context.
- Designed as a dark technical field-manual / public-interface site.

## Deployment

The build output is `dist/`.

For GitHub Pages project hosting, Vite uses `base: './'` so assets work from both a project subpath and a root/custom domain.
